About

Hello! If you hadn't already guessed, my name is Chris Cardimen. I'm a michigan-based software developer currently working at Jackson National Life. I do full-stack development, as well as website design and game development.

This website

This is my personal website, chriscardimen.com. I primarily use it as a portfolio and as a place to document my projects.

Source Code

The source code for this website is all publicly available on GitHub, under the website-chriscardimen repository.

The entire website has been handmade in classic barebones HTML. If you want to see my work with an engine (WordPress) that has more design features, check out the website I made for my wedding: cardimensteffywedding.com. The deployment configuration of this site is done through a combination of Git, cPanel (the hosting site I use), and cron jobs. When a change is merged into the main branch of this repo, cPanel's cron automation suite will send out a cron job to run and execute a PHP script to update the site.
